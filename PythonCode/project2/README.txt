OBJ to VTK converter V 1.0.0 2/26/14
Author: Kevin Dalle

GENERAL USAGE NOTES
------------------------------------

- On the command line the only things that should be entered is ipython project2.py example.obj
- Example is a placeholder for the obj file that you wish to convert to VTK
- Be aware that the program will accept file formats other than obj files, but obj is the only
  files that the program can convert to VTK
- The first output file will be example.vtk, which is the non-Triangulated file, the second out
  put will be example2.vtk which is the Triangulated file
