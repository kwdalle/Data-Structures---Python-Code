Itinerary V 1.0.0 4/22/15

GENERAL USAGE NOTES
----------------------------------------------------------------------------------------------
- You run the program on the command line as ipython project5.py
- You will first enter a origination and then a destination.
- It will then show a list of cities from the origination to the destination
  in the order of the quickest path.
- It will then ask if you would like to plan another trip between two another
  cities.
- If you say no to looking at another two cities it will ask you if you want to
  have the plan written into a Viz graph file.
- The graph Viz file will be called Itinerary.gv
- The Green edge marks the origination while the blue edge marks the destination
  the red lines is the path taken, with all other edges black.
